Lógica De Programação

Loop

Os loops são recursos usados para criar repetições. Se precisamos no nosso programa, realizar uma mesma tarefa várias vezes, ao invés de escrevê-la inúmeras vezes (muitas vezes não sabemos quantas vezes serão necessárias), nós criamos um laço de repetição, um loop, e dentro desse loop, nós escrevemos o que queremos fazer uma únicaa vez, determinando que essa ação se repita x vezes, ou até mesmo, infinitas vezes, até que haja uma interrupção.

Para entender os loops, vamos ver o famoso for loop. For que significa "para", é uma estrutura de repetição muito usada e necessária.

Exemplo de uso do for(para) loop

  • para ( inicio (0) -> aqui você determina o começo do loop ; final (10) -> aqui onde seu loop deve acabar ) {

  • Aqui você escreve o código que você quer que se repita, e ele irá se repetir a quantidade de vezes que você determinou na contagem de 0 (início), até 10 (final), sendo assim, esse código se repete 10 vezes.

  • } -> fechamento do loop

Vamos supor que você precisa fazer um programa que conte até dez, então, ao invés de você escrever o código para escrever o número 1, depois escrever o mesmo código de novo pra escrever o número 2, depois o 3,... Até o 10... Você cria um loop, onde você escreve o código de escrever o número uma única vez, para que ele se repita 10 vezes, mudando o número, começando pelo 1, até chegar no 10.

A estrutura de um loop é diferente em cada linguagem, então não se preocupe se ainda parece meio confuso a maneira de escrever o código de um loop, você irá aprender a escrever e usar o loop nas linguagens que você quiser aprender, esse não é o objetivo agora, por enquanto, apenas queremos que você entenda o conceito de loop.
