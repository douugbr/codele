Introdução à plataforma

HUD

Hoje, aprenderemos sobre o que é uma HUD, e como criar um pequeno exemplo de uma no Game Maker.

A HUD (heads-up display - tela de alerta) é a sigla para representação dos objetos do jogo, tais como: life - do inglês vida, force - do inglês força), magia (às vezes representados por: mana - ou MP, Mana Points -). Esses atributos (assim como são chamados no jogo) variam de nome e de valor de acordo com o jogo (podendo ser traduzido para a língua original do projeto). Pode ser também usada para representar itens e outras objetos do projeto/jogo. Hoje, criaremos uma HUD simples, que mostre na tela apenas os pontos de vida do jogador, a aula de hoje também servirá de porta de entrada para que você possa criar diferentes tipos de HUD.

Para trabalhar com esse conceito no game Maker, usaremos o evento “Draw”. Crie um objeto, e dê a ele o nome de “obj_hud”, em seguida, dentro deste objeto crie um evento “Draw”. Usaremos um código deste evento para mostrarmos a quantidade de vida do jogador na tela, usaremos a função “draw_text” dentro de um if, e a condição para que a função “draw_text” seja executada, é que o objeto do jogador exista na sala. Para isso, usaremos uma função chamada “instance_exists()”, o código ficará assim:

  • if instance_exists(obj_player) {
  • draw_text(x,y,obj_player.hp)
  • }

Este código pode ser lido como:

“Se o obj_player existir: desenhar texto(hp do obj_player)”


Obs: É importante lembrar que não se deve adicionar uma sprite ao “obj_hud”


Agora, basta adicionar o obj_hud no canto de sua room, e assim iniciar o jogo.


Hoje aprendemos um sistema básico de HUD para jogos, na próxima aula, aprenderemos como fazer uma barra de vida personalizada com os conhecimentos aprendidos hoje. Até mais!
