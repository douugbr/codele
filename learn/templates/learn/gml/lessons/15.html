Barra de Vida

Hoje aprenderemos a como criar uma barra de vida simples no Game Maker Studio 2. Primeiramente, precisaremos criar uma variável no “create” do player, para a quantidade de vida que ele terá.

Em seguida, iremos criar um “if”, determinando que quando os pontos de vida do jogador forem iguais o menores que 0, o objeto do jogador seja destruído. O código ficará parecido com isto:

  • if hp <= 0 {
  • instance_destroy()
  • }


Agora, iremos partir para o objeto do inimigo, e apagaremos o código que faz com que o inimigo seja destruído ao entrar em contato com o inimigo, e em vez disso, escreveremos um que fará com que a o jogador perca pontos de vida enquanto estiver colidindo com o inimigo, o código ficará mais ou menos assim:

  • if place_meeting(x,y,obj_player) {
  • obj_player.hp -= 0.3
  • }

Com isto, o inimigo enquanto estiver colidindo com o jogador, tirará pontos de vida dele.


Hoje aprendemos um pouco sobre como funcionam os pontos de vida, e na próxima aula, aprenderemos sobre o que é uma HUD, e como aplicar no nosso jogo. Até mais!
