Criação de Objetos

Na aula de hoje, aprenderemos a criar objetos no Game Maker Studio 2, no projeto em que estamos trabalhando desde o começo das aulas.

Para realizar a ação de criar objetos, primeiramente vamos criar uma sprite e um objeto para o mesmo que iremos criar na room. Depois de darmos um nome ao nosso objeto, e atribui-lo uma sprite, iremos ao código step do nosso player.

Para criar objetos na room, é usada uma função chamada “instance_create()”, no qual dentro dos parêntesis deve ser inserida a localização em x onde o objeto será criado, a localização em y, a deph, e o objeto que deve ser criado.

Iremos escrever um código que faça com que ao apertar a tecla “Z“, nosso objeto seja criado 10 pixels a frente do player. O código ficará parecido com isso:

  • if keyboard_check(ord("Z")) {
  • instance_create_depth(x+10,y,1,obj_1)
  • }

Obs: “obj_1” deve ser trocado pelo nome do objeto no qual você irá criar.


Na aula de hoje aprendemos a criar objetos no Game Maker, na próxima, entenderemos como usar o Alarm, para que possamos adicionar um delay entre o tempo de criação de objetos, ou qualquer tipo de forma de trabalhar com o tempo no game Maker. Até a próxima!
