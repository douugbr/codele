Lógica De Programação

Keywords

Vamos agora falar sobre keywords (palavras-chave). Uma coisa que muitas pessoas podem ficar receosas ao emprender programação, é em relação ao idioma, afinal as linguagens são escritas em inglês, e as pessoas que não sabem muito o idioma sentem medo de acabar não conseguindo entender, mas calma… o inglês é sim algo muito bom em todas a ocasiões, então é bom ter um conhecimento básico, mas preste atenção, não precisa saber inglês para programar, você só precisa entender algumas keywords, que podem ser o nome de uma estrutura condicional por exemplo, você já pode saber o que algumas palvras significam pra já ter um ideia de como ela funciona. Vou mostrar alguns exemplos:

  • If = Se
  • Else = Se não
  • For = para
  • While = enquanto
  • Int (integer) = número inteiro
  • Char (Character) = Caractere
  • Class = Classe
  • Double = Dobro
  • Long = Longo
  • Print = Imprimir
  • Catch = Pegar
  • Break = Quebrar
  • Return = Retorna
  • goto = ir a/para
  • When = Quando
  • Do = Faça
  • Or = ou
  • And = E
  • As = Como
  • Try = Tente
  • Except = Exceto
  • Finally = Finalmente
  • Import = Importar
  • Include = Incluir
  • Export = Exportar
  • Var/Variable = variável
  • With = Com
  • List = Lista
  • From = de
  • Is = é
  • True = Verdadeiro
  • False = Falso

Enfim, essas são algumas das keywords de linguagens variadas, que na verdade, grande parte, não é somente de uma determinada linguagem específica, mas pode ser usada em várias, mas estamos colocando essa lista para você saber o que essas palavras significam, podem ser tipos de variáveis, funções, loops, etc.
