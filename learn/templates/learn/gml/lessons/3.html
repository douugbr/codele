Lógica De Programação

Tipos Primitivos

Na programação temos muitas funcionalidades e ferramentas, temos por exemplo: loops, cujo o objetivo é criar laços de repetição para que uma ação seja repetida tantas vezes quanto forem determinadas, temos bibliotecas que são como um pacote de funcionalidades e propriedades para serem usadas no nosso código, e para usá-las precisamos “trazer” a biblioteca que as possui para o nosso programa, entre outras coisas, mas tudo isso, é algo que você aprenderá mais pra frente no aprendizado de uma linguagem específica também, mas o principal objeto que eu quero falar é sobre variáveis… que são espécies de espaços da memória que podemos criar para guardar informações, é o que usamos para manipular dados, como por exemplo, criamos uma variável nome, com o objetivo de pegar um nome de alguem ou algo e armazenar essa informação para que possamos usar no programa. E o tópico é justamente sobre isso: “informações” e seus tipos… quando você vê em algum código com algum trecho com aspas é porque está se tratando de uma mensagem de texto… Mas texto não é o único tipo de dado que temos, portanto vou listar agora alguns tipos primitivos de dados:

  • Texto: Se trata de uma mensagem de texto, sempre delimitado por aspas, pode ser uma palavra, uma frase, um nome, um texto, uma citação, ou até mesmo um número, contanto que ele esteja ali apenas para ser mostrado e não para fazer cálculos.
    Exemplo: “Computador”, “Lógica de programação é fácil”, “Olá mundo!”, “Agente 46”, “10”.

  • Inteiro: É um dado do tipo “inteiro”, ou seja, um número inteiro, então, qualquer número inteiro é considerado int, quando você planeja usá-lo em algum cálculo, neste caso, não se coloca aspas.
    Exemplo: 5, 12, 16, 145, -8, 500.

  • Real: São números reais, ou seja, com casa decimal, números que tenham “.”, não necessariamente precisa ser um número quebrado.
    Exemplo: 13.0, 5.6, 0000.14, -10.0.

  • Lógico: São o que chamamos de "valores lógicos" que apenas comportam os valores: True (verdadeiro) ou False (Falso)

Claro que podemos ter em determinadas linguagens alguns outros tipos, como por exemplo, em java, temos o tipo byte que é capaz de armazenar valores entre -128 até 127. Enfim, os tipos primitivos de dados são importantes para declarar um tipo de variável, ou um tipo de dado que será recebido através de uma interação com o usuário... De maneira geral, os tipos de dados são importantes para que você possa manipulá-los.
