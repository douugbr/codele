Trabalhando com Animações

Antes de começar animar o nosso jogo, vamos criar as definições: o que é animação? Animações são sequências de imagens que criam a ilusão de movimento. Cada imagem que forma a sequência é chamada de Quadro (ou Frame), tanto que uma qualidade de animação pode ser medida em FPS (Frames Per Seconds). Essa métrica indica quantos quadros são usados para gerar 1 segundo de animação. Por exemplo, muitos filmes possuem FPS de 24, ou seja, 24 imagens para gerar 1 segundo. Jogos costumam estar entre 30 e 60 FPS, porque em filmagens é possível criar efeitos de velocidade com apenas manejo da câmera, mas o mesmo não pode ser facilmente aplicado em jogos.

Vamos trabalhar encima desse conceito de animação, com o projeto que criamos na aula passada. O objetivo, é fazer com que o cubo que foi criado mude de cor quando começar a se mover.

Para isso, é necessário criar uma nova sprite para o cubo, e pintá-la de uma nova cor. Em seguida, iremos para o código “Step” no jogador, e dentro do “if” onde está o código de movimentação do jogador, iremos escrever:

Sprite_index = x

(“x” corresponde ao nome da sprite de nova cor que foi criada).


Desta forma, o código ficará assim:

  • if keyboard_check(vk_right) = true {
  • x+=velocidade
  • sprite_index = x
  • }


Depois de compilar o jogo, é possível perceber que depois de movimentar o cubo, a cor do objeto muda, porém não volta a cor original quando para de se mover. Para resolver isso, iremos voltar ao nosso código, e utilizar a função “Else” depois do “If”, desta forma:

  • else {
  • sprite_index = y
  • }

(“y” corresponde ao nome da sprite original do cubo).


Então, o código inteiro ficará dessa forma:

  • if keyboard_check(vk_right) = true {
  • x+=velocidade
  • sprite_index = x
  • }
  • else {
  • sprite_index = y
  • }


Com este conceito, é possível criar animações de movimento, pulo, ataque, morte, entre diversas outras. Também é possível usar o código “image_speed = x” para definir a velocidade de animação que o objeto terá.


Hoje aprendemos a como trabalhar com animações de forma simples. Na próxima aula, iremos trabalhar com alguns códigos mais avançados. Até mais!
